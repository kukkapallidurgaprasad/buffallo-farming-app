```cmd

```

```cmd
# Drop and recreate the database
psql -U postgres -c "DROP DATABASE buffalo_farming;"
psql -U postgres -c "CREATE DATABASE buffalo_farming;"

# Import new schema with 16 breeds
cd C:\Users\hello\OneDrive\Desktop\26-dec-2025\buffalo-farming-app
psql -U postgres -d buffalo_farming -f database\schema.sql
```

### **Step 3: Restart Backend**
```cmd
cd backend
python app.py
```

- Go to browser
- Press `CTRL + F5` (hard refresh)
- Go to Breeds tab
- Should now see **16 breeds** instead of 7!
